## 🎯 功能特性

### 核心甘特圖功能
- **級聯里程碑** - ID Design → Fine Model → BU Review → PPM → ER
- **工作日計算** - 自動排除週末和自訂假日（台灣/中國節假日）
- **檢查點標記** - CP1、CP2、CP3、CP4 關鍵節點
- **工作日編號** - 顯示 WD (Working Day) 序號
- **月份分割線** - 月末和星期日視覺標記

### 交互功能
- **行拖拽重排** - 平順動畫的行順序調整（FLIP 動畫技術）
- **列凍結** - 左側行標籤在水平捲動時始終可見（sticky 定位）
- **水平縮放** - 30% 至 100% 的 10 級縮放滑塊
- **行標題雙擊編輯** - 快速修改行名稱

### 色彩管理
- **雙色彩選擇器** - 分別控制條形圖和背景色
- **HSB 滑塊選擇器** - 色相、飽和度、亮度精確調整
- **最近使用色** - 確認後保存常用顏色（最多 10 個）
- **灰階快速選擇** - 預設灰度色板

### 條形圖操作
- **邊緣調整** - 拖拽左右邊緣改變條形日期範圍（ew-resize 游標）
- **長按移動** - 1.5 秒長按進入移動模式，自由拖拽條形
- **快速點擊菜單** - 4px 拖拽臨界值區分點擊和拖拽

### 行管理
- **行複製** - 完整複製行內容、顏色、邊框、批次設定
- **行刪除** - 帶有警告對話框的破壞性操作
- **邊框樣式** - 粗邊框（3px）或細邊框（1px）選擇

### 進階功能
- **X 輪批次標籤** - 設置批次輪數和段落標籤
  - 例：「2 輪」會分成 2 段，自動標記為 1、2
  - 標籤顯示在條形圖內部
- **手動單元格編輯** - 拖拽選擇範圍自動建立條形圖
  - 不需確認對話框，立即編輯
  - Esc 取消，Enter/Blur 提交

### 狀態持久化
- **自動保存** - 所有更改儲存在瀏覽器 localStorage
- **完整恢復** - 重新載入頁面後完全復原狀態
  - 條形圖位置（startDate/endDate override）
  - 行邊框樣式
  - 隱藏行狀態
  - 最近使用色清單

## 🚀 快速開始

### 使用方法
1. 在瀏覽器中打開 `index.html`
2. 左側面板調整參數和設定
3. 中央區域查看和交互甘特圖
4. 右側浮窗顯示詳細資訊

### 快捷操作
| 操作 | 方法 |
|------|------|
| **調整行順序** | 拖拽左側 ≡ 菜單按鈕 |
| **縮放圖表** | 使用頂部縮放滑塊 (30%-100%) |
| **編輯行標題** | 雙擊行標籤 |
| **調整條形日期** | 拖拽條形左/右邊界 |
| **移動條形** | 長按（1.5s）條形後拖拽 |
| **編輯單元格** | 拖拽選擇日期範圍 |
| **行操作菜單** | 按下 ≡ 按鈕（複製/邊框/刪除） |

## 🛠 技術棧

- **HTML5** - 語義標記

- **HSV↔RGB 色彩空間** - 精確的色彩轉換
- **Sticky 定位** - 凍結列不使用 JavaScript 捲動監聽
- **狀態序列化** - JSON 格式持久化和恢復

## 💾 狀態管理

應用狀態在 localStorage 中自動持久化：

```javascript
{
  zoom: 0.7,                    // 縮放級別 (0.3-1.0)
  rowOrder: [...],              // 行顯示順序
  rowColors: {...},             // 行顏色覆蓋
  bgColors: {...},              // 背景色覆蓋
  barOverrides: {...},          // 條形圖日期覆蓋
  rowBorders: {...},            // 邊框樣式
  hiddenRows: [...],            // 隱藏行
  recentColors: [...],          // 最近使用色
  ...
}
```
